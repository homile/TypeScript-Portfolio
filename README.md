# TypeScript-Portfolio
나만의 포트폴리오 사이트 (TS.ver)

## 목적   
타입스크립트를 공부하며 이전 바닐라JS로 작업한 포트폴리오 사이트를 React로 마이그레이션 해보기 위함.

> 배포링크:
